Detailed Description

Force structure packing at 4 byte alignment.

This shouldn't be nested – included it around code only.

If you don't like ugly C preprocessor code, don't look at this file. :)

This is necessary if the header is included in code which has structure packing set to an alternate value, say for loading structures from disk. The packing is reset to the previous value in close_code.h
